# The LLM Training Pipeline

> The three-stage journey: pretraining → supervised fine-tuning → reinforcement learning from human feedback.

## Key Takeaways

- **Pretraining** — self-supervised learning on massive text corpora; learns language, facts, and world knowledge
- **Supervised Fine-Tuning (SFT)** — teaches the model structure and behavior through high-quality examples
- **RLHF (Reinforcement Learning from Human Feedback)** — optimizes for human preferences using a reward model
- **Two-phase view** — pretraining (builds capabilities) + post-training (refines and aligns)
- **Critical insight** — understanding the full pipeline is essential; each stage builds on the previous
